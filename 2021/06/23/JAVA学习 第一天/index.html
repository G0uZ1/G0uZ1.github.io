# 需要注意 的问题

  • Java代码对大小写敏感，如果存在大小写的错误，程序是无法运行的。
  • 代码中//为程序的注释符，方便读者检查理解代码的功能或意义。
  • 关键字public 即访问修饰符是（access modifier）,它是用来控制程序其他部分对控制代码部分的访问级别。
  • 关键字class 表明Java程序中的全部内容都包含在类中，类是作为一个加载程序的逻辑容器。
  • 关键字class之后紧跟着的是类名。Java中定义的规则不是很严格，名字必须以字母开头，后面可以跟字母和数字的任意组合，长度基本没有限定，但时不能使用Java保留字。
  • Java类名应该符合标准的命名规范：类名是以大写字母开头的名词。如果名字由多个单词构成，则每个单词的第一个字母都必须大写。
  • 在类名Demo之后对应的半个花括号{，与最后一行的}对应。做了类声明的开始和结束符号。

JDK包中的常用命令

程序 说明
javac java文件编译器，用于将java源代码（*.java）编译问字节码（*.class）。
java java解释器，解释并执行 *.class 文件。
jdb java调试工具，帮助程序员发现程序中的错误，以及对错误进行定位和跟踪。
javap java反编译器，将java字节码（*.class）反编译为源代码。
